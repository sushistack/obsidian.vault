
> Distribute Document Store

가장 중요한 Idea는 **Inverted Index**(역색인) 이다.  Inverted Index 란 document 에 있는 단어들을 나열하고 각 단어가 나오는 document 를 식별 하는 방식을 말한다.

즉, 단어 검색 시  O(1)의 시간 복잡도를 갖는다.


| Type   | Data structure |
| ------ | -------------- |
| Text   | Inverted Index |
| Number | BKD Tree       |
| Geo    | BKD Tree       |
