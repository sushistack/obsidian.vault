## 함께 쓰이는 말 들

- so for
- since N / S + V
- for (~ 숫자) / in

1. So far, (it's been) so good.
2. We've spent 300,00 won so far.    (until now -> 지금까지는.. 반전의 느낌을 줄 수 있음)
3. I've been busy since Monday.
4. I've had a crush on her since I first saw her.
5. I haven't drunk in 2 months.
6. I haven't seen him in a while. (한 동안)
