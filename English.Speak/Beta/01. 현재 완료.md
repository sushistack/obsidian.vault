경험이 있는 상태를 말할 때 사용한다.

- I have seen that. / I've seen that.
- I have not been there / I haven't been there.
- She has left. / She's left.
- Have we met before? / Yes, we have. / No, we haven't
- I've done it before.
- Have we meet?
- Have we ever met? (ever = 경험을 묻는 것을 확실하게 만들어주는 느낌)

### 완료된 동작

과거로 써도 되지만, 현재 완료된 상태를 강조.

1. I've cleaned the living room.
2. We've done our duty. Andy's grown up.
3. I've decided what to buy.
4. It has already begun.

### 새로운 소식 전달

생생한 느낌을 준다? 단순 과거로 쓰면 5년 전인지 방금 전인지 모른다.

1. There has been an accident.
2. There has been a mistake.
