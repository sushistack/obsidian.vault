
# 📁 AI 콘텐츠 비즈니스 아키텍처: [The Retro Archives]

## 1. 🎯 전략적 포지셔닝: "불완전함의 완벽함"

- **핵심 컨셉:** 1980~90년대의 실제 아카이브(Archive)에서 발견된 듯한 '저화질 고몰입' 영상.
    
- **차별화 전략:** 최신 AI(Veo3, Sora)의 고화질 생성 능력을 일부러 **'레트로 필터링'**하여 AI 특유의 이질감을 '아날로그 노이즈'로 승화시킴.
    
- **비즈니스 모델:** 유튜브 광고 수익 + 글로벌 라이선싱(Scary Stock) + Patreon 기반 미공개 기밀 문서 구독.
    

---

## 2. 🛠 기술 스택 고도화 (Technical Pipeline)

### [A] 시각화: Realistic-Animate 하이브리드

- **Character Consistency:** ComfyUI의 `Realistic to Animate` 워크플로우를 통해 실사 캐릭터의 일관성을 유지.
    
- **Backgrounds:** **Veo3 / Sora**를 활용하여 8090년대의 도시, 연구소, 사건 현장을 시네마틱하게 생성.
    
- **Retro Layering:** 생성된 4K 영상을 ComfyUI의 `Animate2Realistic` 노드에서 의도적으로 **Downscaling → Noise Injection → Chromatic Aberration** 처리하여 VHS 질감 구현.
    

### [B] 오디오: Qwen3-TTS + Google Mixboard

- **Voice:** **Qwen3-TTS**를 이용해 90년대 성우/아나운서의 목소리를 클로닝. 감정 조절의 한계는 **n8n** 단계에서 문장 사이에 `[breath]`, `[...]` 등의 토큰을 삽입하거나, **Google Mixboard**를 통해 배경음과 음성의 밸런스를 조절하여 긴박감을 보완.
    
- **Music:** Google Mixboard를 활용하여 80년대 신시사이저 기반의 Lo-fi 공포 배경음악 자동 생성.
    

### [C] 자동화: n8n & Backend Logic

- **Workflow:** 1.  **Input:** 미제 사건/SCP 원문 데이터.
    
    2.  **LLM (Grok/Gemini):** 대본화 및 영상 생성을 위한 프롬프트 분해.
    
    3.  **Qwen3-TTS:** 음성 생성.
    
    4.  **ComfyUI/Sora API:** 영상 에셋 생성.
    
    5.  **Final Stitching:** n8n을 통해 음성과 영상을 결합하고 유튜브 예약 업로드까지 파이프라인화.

## 4. 📈 지속 가능성을 위한 고도화 방안 (Scalability)

### 1) 글로벌 자동 번역 파이프라인

- 한국어 대본이 완성되면 n8n을 거쳐 **영어/일본어**로 자동 번역.
    
- Qwen3-TTS의 다국어 기능을 이용해 동일한 목소리 톤으로 일본어/영어 채널 동시 운영 (1인 3개 채널 운영).
    

### 2) 인터랙티브 세계관 (RAG 결합)

- 시청자가 댓글로 "특정 SCP의 뒷이야기가 궁금하다"고 하면, **n8n + Vector DB**에 저장된 세계관 데이터를 바탕으로 AI가 자동으로 답변 영상을 제작하는 '커뮤니티 소통형' 콘텐츠 도입.
    

### 3) 비용 최적화 (9060 XT 활용)

- 고해상도 메인 영상은 **Sora/Veo3** API를 쓰되, 반복적인 캐릭터 컷이나 배경 노이즈 영상은 로컬의 **9060 XT**에서 ComfyUI로 생성하여 API 비용 절감.
